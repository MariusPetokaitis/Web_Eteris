INSTALLATION
------------
1. Upload the 'Forest' directory to your server in the 'Themes' directory.
2. Log on to your website.
3. Go to the Administration-Panel.
4. Go to the menu System Admin.
5. Go to Main Settings.
6. Scroll down and set the setting Site Theme to color.
